# Algoritmo: Array Exercises

Este algoritmo consiste em uma atividade desenvolvida em sala de aula, tendo como principal objetivo a utilização dos conhecimentos adquiridos, como:

## Conceitos utilizados:
- Manipulação de **arrays** (vetores)
- Estruturas de repetição (`for`, `do-while`)
- Estruturas condicionais (`if`, `switch-case`)
- Manipulação de entrada de dados com `Scanner`
- Métodos de ordenação (`Arrays.sort()`, `Collections.reverseOrder()`)
- Operador módulo (`%`) para identificar pares e ímpares
- Modularização de código com métodos estáticos

Neste projeto Java contém exercícios de manipulação de arrays, separação de dados e ordenação, utilizando a classe `Scanner` para interação com o usuário.

## Conteúdo do projeto

O projeto possui um menu com dois exercícios principais:

### 1. **Números Pares e Ímpares**
   - Solicita ao usuário a quantidade de números que deseja digitar
   - Permite a entrada de vários números inteiros
   - Separa e exibe os números **pares**
   - Separa e exibe os números **ímpares**
   - Informa a quantidade total de cada tipo

### 2. **Ordenação de Nomes**
   - Solicita ao usuário a quantidade de nomes que deseja digitar
   - Permite a entrada de vários nomes
   - Exibe os nomes digitados (ordem original)
   - Exibe os nomes em **ordem crescente** (A-Z)
   - Exibe os nomes em **ordem decrescente** (Z-A)

## Como executar

### 1. Clone o repositório:
```bash
git clone https://github.com/fagundessana/array-exercises.git
```

### 2. Navegue até o diretório:
```bash
cd array-exercises
```

### 3. Compile o código:
```bash
javac com/mycompany/arrayexercises/ArrayExercises.java
```

### 4. Execute o programa:
```bash
java com.mycompany.arrayexercises.ArrayExercises
```
